"Coding"

Coding is what makes it possible for us to create computer software, apps and websites.

Code is a set of rules or instructions that tell the computer what to do.

There are literally 1000s of programming languages.

♥ HTML/CSS ♥

Frontend vs Backend

Front end

  • What you see
  • Design, aesthetics, content
  • HTML, CSS, JavaScript

Back end

  • What you don't see
  • Server, database, application
  • PHP, Ruby, Python, C#, Node.js

The most common frontend languages are HTML, CSS and JavaScript.

HTML

A website's text, images and content, plus some simple formatting.

CSS

The styling and design of a website. Controls how the HTML content is displayed.

JavaScript

Adds interactivity to a website. Allows you to make the website behave in different ways.
